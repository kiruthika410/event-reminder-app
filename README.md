
A web-based Event Reminder System developed using Node.js, Express, and Azure DevOps for event scheduling, reminders, agile planning, testing, and CI/CD pipeline integration.
# Event Reminder app

## Overview
The Event Reminder System is a web-based application designed to help users organize and manage events efficiently. The application allows users to create, view, update, and manage reminders.

## Features
- Add and manage events
- Event scheduling with date and time
- Reminder management
- User authentication
- REST API support
- Azure DevOps CI/CD integration
- Load testing support

## Technologies Used
- Node.js
- Express.js
- HTML
- JavaScript
- Azure DevOps
- GitHub

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/event-reminder-system.git
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
node server.js
```

4. Open browser

```text
http://localhost:3000
```

## CI/CD Pipeline
Azure DevOps pipeline is configured using `azure-pipelines.yml`.
